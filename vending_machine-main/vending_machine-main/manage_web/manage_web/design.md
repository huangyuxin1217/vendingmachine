三、界面交互
- 操作确认：删除、修改库存、作废订单等需弹出确认框
- 分页加载：商品/订单列表支持分页，提升性能
- 图表交互：图表可放大、导出 PNG 或 Excel
- 实时监控：售货机状态页可设置自动刷新（如每30秒）
